# autMan 插件集合

这是一个autMan插件的集合仓库,包含我为autMan平台开发的各种实用插件。

## 📦 插件列表

### 1. 肚子疼记录插件 (Stomach Pain Tracker)

> 一个爸爸为了记录孩子肚子疼情况而开发的插件

**功能**: 快速记录和统计孩子肚子疼的时间,帮助父母了解规律,就医时提供准确病史。

**版本**: v1.4.0

**文档**: [查看详情](./stomach_pain/README.md)

**特性**:
- 📝 一键记录时间
- 📊 时间轴视图展示
- 📈 智能分页和月度摘要
- 🎨 彩色频次标记

---

### 2. 奥力给记录插件 💪

> 用于记录每次拉屎的时间戳，帮助你追踪肠道健康

**功能**: 快速记录拉屎时间，支持类型分类(通畅/费劲/拉稀)，可视化统计分析，帮助了解肠道规律。

**版本**: v1.6.1

**文档**: [查看详情](./aoligei/README.md)

**特性**:
- 🕐 自动记录时间戳
- 💩 类型分类标记
- 📊 时间轴视图展示
- 📈 智能分页和月度摘要
- 🎨 彩色频率标记
- 🗑️ 支持单条删除

---

### 3. 体重记录插件 ⚖️

> 完整的体重管理工具,支持记录、趋势分析和目标管理

**功能**: 记录体重数据、补录历史、查看趋势、设定目标体重、智能统计分析。

**版本**: v1.0.0

**文档**: [查看详情](./weight_tracker/README.md)

**特性**:
- 📝 记录体重(当前/历史补录)
- 📊 趋势分析(最高/最低/平均)
- 🎯 目标体重管理
- 📈 7天/30天/全部统计
- ✏️ 数据管理(删除/修改)
- 🎉 达成目标提示

---
