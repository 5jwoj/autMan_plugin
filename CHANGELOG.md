# 更新日志

## v1.1.0 (2026-01-09) ⭐ 重大更新

### ✨ 新功能

- **全新的记录显示样式**: 完全重新设计了"肚子疼记录"的显示格式
  - 📅 按日期分组显示
  - 🎨 颜色标记：🟢 1次/天，🟡 2次/天，🔴 3次及以上
  - 📊 添加详细的统计信息
    - 记录时段范围
    - 记录天数和跨度
    - 总计次数
    - 平均频率
  - 📄 智能分页：最多显示最近10天，避免消息过长
  - 🎯 更美观的排版和分隔线

## v1.0.1 (2026-01-09)

### 🐛 Bug 修复

- **修复 API 调用错误**: 修正了 `getUsername()` 方法不存在的问题
  - 移除了 `self.sender.getUsername()` 调用
  - 改为使用 `self.user_id` 作为用户名
  - 这样可以避免 middleware API 兼容性问题

### 📝 说明

根据实际测试，middleware 的 `Sender` 对象可能没有 `getUsername()` 方法。
现在插件使用用户 ID 作为用户名，确保兼容性。
