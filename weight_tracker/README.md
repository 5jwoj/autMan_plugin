# 体重记录插件 (Weight Tracker)

一个功能完善的autMan体重记录插件,支持体重记录、趋势分析、目标管理等功能。

## 功能特性

### 📝 体重记录
- **当前记录**: 发送 `体重 65.5` 记录当前体重
- **历史补录**: 发送 `体重记录 2026-01-01 65.5` 补录历史数据
- **自动时间戳**: 自动记录每次测量的时间
- **智能更新**: 单日多次记录自动保留最新值

### 📊 数据展示
- **时间轴视图**: `体重记录` - 查看最近7天的体重变化趋势
- **详细列表**: `体重详细记录` - 查看所有带编号的完整记录
- **统计分析**: `体重统计` - 查看全部数据的统计信息
  - `体重统计 7` - 最近7天统计
  - `体重统计 30` - 最近30天统计
- **趋势标识**: 显示体重变化趋势 (↑上升 / ↓下降 / →持平)
- **关键指标**: 自动显示最高、最低、平均体重

### 🎯 目标管理
- **设置目标**: `设置目标体重 60` - 设定目标体重
- **进度跟踪**: `目标进度` - 查看距离目标的差距
- **达成提示**: 达到目标体重时自动显示庆祝提示 🎉

### ✏️ 数据管理
- **删除记录**: `删除体重记录 3` - 删除第3条记录
  - 支持查看详细记录后直接发送编号删除
- **修改记录**: `修改体重记录 3 66` - 将第3条记录修改为66kg
- **清空数据**: `清空体重记录` - 清空所有记录(保留目标体重)

## 使用指南

### 基础使用

1. **开始记录体重**
   ```
   体重 65.5
   ```
   系统会记录当前日期和体重值。

2. **补录历史数据**
   ```
   体重记录 2025-12-25 64.8
   ```
   可以补录任意日期的历史体重数据。

3. **查看记录和趋势**
   ```
   体重记录
   ```
   显示最近7天的体重变化,包含趋势标识。

4. **查看统计信息**
   ```
   体重统计
   ```
   显示最高、最低、平均体重和总体变化。

### 目标管理

1. **设定目标体重**
   ```
   设置目标体重 60
   ```

2. **查看进度**
   ```
   目标进度
   ```
   系统会显示当前体重、目标体重以及还需增重/减重的数值。

### 数据管理

1. **查看所有记录**
   ```
   体重详细记录
   ```
   显示所有记录及其编号。

2. **删除特定记录**
   ```
   删除体重记录 3
   ```
   或查看详细记录后直接发送 `3`

3. **修改记录**
   ```
   修改体重记录 3 66.5
   ```

4. **清空所有记录**
   ```
   清空体重记录
   ```

## 数据格式

插件使用以下JSON格式存储数据:

```json
{
  "records": [
    {
      "date": "2026-01-03",
      "weight": 65.5,
      "timestamp": 1735891200000
    }
  ],
  "target": 60.0
}
```

## 完整命令列表

| 命令 | 功能 |
|------|------|
| `体重 [数值]` | 记录当前体重 |
| `体重记录 [日期] [数值]` | 补录历史数据 |
| `体重记录` | 查看最近记录 |
| `体重详细记录` | 查看带编号的记录 |
| `体重统计` | 查看完整统计 |
| `体重统计 [天数]` | 查看指定天数统计 |
| `设置目标体重 [数值]` | 设定目标体重 |
| `目标进度` | 查看目标进度 |
| `删除体重记录 [编号]` | 删除指定记录 |
| `修改体重记录 [编号] [数值]` | 修改指定记录 |
| `清空体重记录` | 清空所有记录 |
| `体重帮助` | 显示帮助信息 |

## 注意事项

- 体重数值支持小数,如 `65.5`
- 日期格式必须为 `YYYY-MM-DD`,如 `2026-01-03`
- 单日多次记录会自动保留最新值
- 清空记录时会保留目标体重设置
- 体重范围限制为 0-500kg
