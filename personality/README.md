# MBTI性格测试插件

[![版本](https://img.shields.io/badge/version-1.2.0-blue.svg)](性格测试.py)
[![语言](https://img.shields.io/badge/language-Python-green.svg)](性格测试.py)

基于MBTI理论的性格测试插件,通过16道精选问题帮助用户了解自己的性格类型,支持查询任意MBTI类型的详细解释。

## ✨ 功能特性

- ✅ **科学测试**: 基于MBTI四维度理论
- ✅ **快速便捷**: 仅需16道题,3-5分钟完成
- ✅ **详细分析**: 提供16种性格类型的完整描述
- ✅ **历史记录**: 保存测试历史,追踪性格变化
- ✅ **交互友好**: 支持中途退出和超时保护
- ✅ **类型查询**: 直接发送MBTI类型查看详细解释

## 🎯 MBTI简介

MBTI(Myers-Briggs Type Indicator)是一种性格分类理论,包含4个维度:

| 维度 | 含义 | 说明 |
|------|------|------|
| **E/I** | 外向/内向 | 能量来源：从外部世界还是内心世界获取能量 |
| **S/N** | 感觉/直觉 | 信息获取：关注具体细节还是整体概念 |
| **T/F** | 思考/情感 | 决策方式：基于逻辑分析还是情感价值 |
| **J/P** | 判断/感知 | 生活方式：喜欢计划性还是灵活性 |

组合这4个维度,形成16种性格类型:

### 🔬 分析师
- **INTJ** - 建筑师: 富有想象力和战略性思维
- **INTP** - 逻辑学家: 创新思维,热爱理论
- **ENTJ** - 指挥官: 天生的领导者
- **ENTP** - 辩论家: 思维敏捷,善于辩论

### 💚 外交官
- **INFJ** - 提倡者: 理想主义,富有洞察力
- **INFP** - 调停者: 理想主义,富有创造力
- **ENFJ** - 主人公: 富有魅力的领导者
- **ENFP** - 竞选者: 热情洋溢,充满活力

### 🛡️ 守护者
- **ISTJ** - 物流师: 务实可靠,注重细节
- **ISFJ** - 守卫者: 温暖体贴,乐于助人
- **ESTJ** - 总经理: 高效务实的管理者
- **ESFJ** - 执政官: 热心助人,善于合作

### 🎨 探险家
- **ISTP** - 鉴赏家: 动手能力强,善于解决问题
- **ISFP** - 探险家: 艺术气质,审美能力强
- **ESTP** - 企业家: 精力充沛,行动派
- **ESFP** - 表演者: 热情开朗,充满活力

## 📝 使用方法

### 基本命令

```
性格测试          # 开始新的性格测试
性格测试记录      # 查看历史测试记录
性格测试删除      # 删除指定的历史记录
性格测试帮助      # 显示帮助信息
INTJ/ISFJ等      # 查询指定MBTI类型的详细解释
```

### 测试流程

1. **开始测试**: 发送"性格测试"
2. **确认开始**: 输入 y 确认,n 取消,q 退出
3. **回答问题**: 依次回答16道题,每题选择 A 或 B
4. **查看结果**: 获得性格类型和详细分析
5. **保存记录**: 自动保存到历史记录

## 🚀 安装步骤

### 1. 下载插件

```bash
wget https://raw.githubusercontent.com/5jwoj/autMan_plugin/main/personality/性格测试.py
```

### 2. 上传到autMan

```bash
scp 性格测试.py root@服务器:/root/aut/plugin/scripts/
```

### 3. 设置权限

```bash
chmod 755 /root/aut/plugin/scripts/性格测试.py
```

### 4. 重启autMan

```bash
systemctl restart autman
```

## 💡 使用建议

1. **第一直觉作答**: 不要过度思考,选择最符合直觉的选项
2. **真实回答**: 选择真实的自己,而非理想中的自己
3. **多次测试**: 可以在不同时期测试,观察性格变化
4. **参考价值**: MBTI仅供参考,不能完全定义一个人

## ⚠️ 注意事项

- 本测试为简化版(16题),完整MBTI测试通常需要60-93题
- 测试结果仅供参考,不应作为重要决策的唯一依据
- 性格类型可能随时间和经历发生变化
- 每个人都是独特的,不应被简单归类
