# 肚子疼记录插件 - 使用说明

## 📝 功能介绍

这是一个autMan插件,用于自动记录和统计孩子肚子疼的时间。

## 🚀 快速开始

### 安装

1. 将插件文件放到autMan的replies目录:
   ```bash
   /root/autman/plugin/replies/肚子疼.js
   ```

2. 重启autMan

### 使用

| 指令 | 功能 |
|------|------|
| `肚子疼` | 记录当前时间 |
| `肚子疼记录` | 查看所有记录 |
| `清空肚子疼记录` | 清空所有记录 |
| `肚子疼帮助` | 显示帮助信息 |

## 🔧 技术说明

- **数据存储**: 使用autMan的bucketSet/Get系统
- **存储桶**: `stomach_pain`
- **数据隔离**: 每个用户的数据独立存储
- **数据格式**: JSON
